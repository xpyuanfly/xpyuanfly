## 👨‍🏫 基本简介

袁鑫攀，湖南工业大学计算机与人工智能学院副院长，本、硕、博均毕业于中南大学计算机相关专业。长期从事工业智能、医学图像分析与多模态信息检索方向研究，围绕**工业物联网智能运维**、**基金项目文本/图像相似性检测**、**鼻内镜诊断报告生成**等应用场景，形成了较为系统的研究与应用成果。

主持国家自然科学基金、湖南省自然科学基金（面上、青年、医卫联合）、教育厅科技计划等国家级与省部级科研项目10余项，作为核心技术负责人研制的《国家自然科学基金相似性检测系统》已在实际业务中广泛应用，实现文本、图像和引文多模态相似性检测，相关成果转化数十项，获教育部科技进步奖一等奖。  

担任工业智能运维株洲市重点实验室主任、工业数据智能湖南省工程技术研究中心副主任，作为科技特派员牵头合作企业建设“湖南省工业物联网云平台”，获工信部 APP 优秀解决方案奖、湖南省科技进步二等奖等。  

在 AAAI、EMNLP、MICCAI、ICME、ICASSP 等国际会议及《Genetics in Medicine》《IEEE Transactions on Instrumentation and Measurement》等权威期刊发表论文二十余篇（第一/通讯作者），申请/授权发明专利十余项，软件著作权多项。长期指导学生参加“互联网+”、计算机设计大赛、蓝桥杯等竞赛，获得国家级、省级奖项三十余项。
